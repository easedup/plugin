Create a learning activity for: $ARGUMENTS

## Steps

1. **Identify context** — determine subject, year level, and purpose of the activity from the arguments
2. **Search curriculum** — if creating for a specific curriculum outcome, use EasedUP MCP to find the relevant content descriptions
3. **Select learning modes** — choose appropriate modes (collaboration, discussion, experiential, guided, independent, etc.)
4. **Generate the resource** — write the complete activity as TipTap-compatible HTML

## Required Sections

- **Activity Title**
- **Subject Area**
- **Year Level**
- **Curriculum Content Descriptions** — from EasedUP MCP if applicable
- **Learning Intention** — what students will know or be able to do
- **Success Criteria** — specific, measurable outcomes
- **Pedagogical Purpose** — rationale for why this activity supports learning (which strategies/modes it addresses)
- **Duration** — estimated time
- **Student Grouping** — individual / pairs / small groups / whole class (with rationale)
- **Materials** — all resources, equipment, technology needed
- **Instructions** — clear, step-by-step student-facing instructions
- **Differentiation** — adjustments for students needing extension or additional support
- **Assessment** — how student learning will be assessed during or after the activity

Output the complete activity as HTML.
