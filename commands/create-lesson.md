Create a detailed lesson plan for: $ARGUMENTS

## Steps

1. **Identify the curriculum context** — determine subject, year level, and curriculum framework (Australian Curriculum v9, Common Core, etc.) from the arguments or ask if unclear
2. **Search curriculum standards** — use the appropriate EasedUP MCP tool to retrieve relevant content descriptions and achievement standards for this topic and year level
3. **Plan the lesson structure** — select 2–3 appropriate teaching strategies from the teaching strategies guide
4. **Generate the resource** — write the complete lesson plan as TipTap-compatible HTML

## Required Sections

The lesson plan must include all of the following:

- **Lesson Title**
- **Subject Area**
- **Year Level**
- **Curriculum Content Descriptions** — quoted accurately from EasedUP MCP results, with content description codes
- **Learning Intentions** — what students will know, understand, and be able to do
- **Success Criteria** — specific, measurable indicators of achievement
- **Prior Knowledge** — what students need before this lesson; how to activate it
- **Assessment** — formative and/or summative strategies; how feedback will be given
- **Materials** — all resources, technology, and equipment needed
- **Lesson Procedure**
  - *Introduction* — engaging hook, connecting new and prior knowledge
  - *Development* — teaching strategies, activities, modelling, guided practice, questioning
  - *Conclusion* — consolidation, reflection, checking for understanding
- **Differentiation** — adjustments for diverse learners (extension and support)
- **Reflection** — prompts for teacher reflection after the lesson

Output the complete lesson plan as HTML.
