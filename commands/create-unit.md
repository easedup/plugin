Create a unit of work for: $ARGUMENTS

## Steps

1. **Gather key details** — if not provided, ask for:
   - Number of weeks and lessons per week
   - Year level and subject/learning area
   - Central theme or driving question (if any)
   - Curriculum framework (Australian Curriculum v9, Common Core, etc.)
2. **Search curriculum standards** — use the appropriate EasedUP MCP tool to retrieve relevant content descriptions and achievement standards for the unit scope
3. **Design the sequence** — plan a coherent progression of lessons with clear learning build-up
4. **Generate the resource** — write the complete unit of work as TipTap-compatible HTML

## Required Sections

- **Unit Title**
- **Subject Area**
- **Year Level**
- **Duration** — total weeks, lessons per week, approximate hours
- **Primary Theme / Driving Question**
- **Curriculum Content Descriptions** — quoted from EasedUP MCP, with codes
- **Achievement Standards** — from EasedUP MCP
- **Learning Intentions** — what students will know, understand, and be able to do across the unit
- **Success Criteria** — clear and measurable indicators
- **Assessment Tasks** — both formative (ongoing) and summative (end-of-unit), aligned to learning intentions; include rubric guidance
- **Learning Activities Sequence** — week-by-week or lesson-by-lesson overview with teaching strategies
- **Practice Principles** — relevant pedagogical principles (high expectations, inclusion, student voice, deep learning, feedback, etc.)
- **Teaching Strategies** — 2–3 key strategies with rationale
- **Resources** — materials, technology, external resources
- **Differentiation Strategies** — for diverse learners
- **Evaluation** — how the unit's effectiveness will be measured (student feedback, assessment data, teacher reflection)

Output the complete unit of work as HTML.
