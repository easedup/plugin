Create a factsheet for: $ARGUMENTS

## Steps

1. **Identify the topic scope** — determine what key concepts and facts need to be covered for this year level
2. **Search curriculum** — use EasedUP MCP to identify the essential knowledge and understanding for this topic
3. **Curate, don't explain** — select facts, definitions, and figures; do not write lengthy explanations
4. **Generate the resource** — write the complete factsheet as TipTap-compatible HTML, maximum 1–2 pages

## Required Sections

- **Factsheet Title** — clear and specific (e.g., "Fractions — Year 5 Quick Reference")
- **Subject Area · Year Level**
- **Key Concepts** — the 5–10 most important ideas, facts, or definitions students need
- **Content Format** — choose the clearest format for the content:
  - Bullet point lists for facts and definitions
  - Short tables for comparisons or data
  - Numbered steps for processes
  - Definition boxes for key vocabulary
- **Examples** — 1–2 worked examples where helpful (e.g., maths procedures, grammar rules)

## Quality Guidelines

- **Core concepts only** — no lengthy explanations; this is a quick reference, not a textbook
- **Student-facing language** — appropriate for the year level
- **Scannable** — use headings, bullets, and tables to make it easy to navigate
- **Maximum 1–2 pages** — tight and focused
- **Differentiation** — if appropriate, note where a simpler or more advanced version could be created

Output the complete factsheet as HTML.
