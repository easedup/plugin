## Strategy vs Activity

| | Teaching Strategy | Learning Activity |
|---|---|---|
| **Scope** | Broad, long-term approach | Specific, short-term task |
| **Duration** | Spans multiple lessons | Minutes to hours |
| **Planning level** | Unit/curriculum | Lesson level |
| **Examples** | Collaborative Learning, Explicit Teaching | Think-Pair-Share, Exit Ticket, Gallery Walk |

## Core Teaching Strategies

1. **Setting Goals** — SMART learning outcomes; best for unit planning, long-term projects, assessment design
2. **Structuring Lessons** — Coherent sequences with timing and cognitive load management; best for daily lessons and workshops
3. **Explicit Teaching** — Direct instruction with clear explanation and modelling; best for new concepts, complex procedures, foundational knowledge
4. **Worked Examples** — Step-by-step demonstrations with reasoning; best for maths problems, scientific procedures, technical processes
5. **Collaborative Learning** — Structured group work with positive interdependence; best for project-based learning, problem-solving, peer teaching
6. **Multiple Exposures** — Varied encounters with content over time; best for vocabulary, concept reinforcement, language learning
7. **Questioning** — Strategic questions at different cognitive levels with wait time; best for discussion, critical thinking, comprehension checking
8. **Feedback** — Timely, specific, actionable feedback creating a learning dialogue; best for formative assessment, writing improvement, skill development
9. **Metacognitive Strategies** — Teaching students to plan, monitor, and evaluate their own thinking; best for study skills, self-assessment, research projects
10. **Differentiated Teaching** — Multiple paths to learning based on readiness, interest, and profile; best for mixed-ability and inclusive classrooms
11. **Classroom Discussion** — Structured verbal interaction (Socratic seminars, fishbowl, accountable talk); best for critical analysis, perspective sharing
12. **Self-Regulated Learning** — Developing student autonomy through planning and reflection strategies; best for independent learning and goal setting
13. **Deliberate Practice** — Focused repetition with immediate feedback and incremental challenge; best for skill mastery, technical proficiency
14. **Design Thinking** — Iterative human-centred problem-solving (empathise → define → ideate → prototype → test); best for innovation projects
15. **Reciprocal Teaching** — Students lead discussions using predict, question, clarify, summarise; best for reading comprehension and text analysis
16. **Visible Thinking Routines** — Scaffolded procedures to make thinking visible and discussable; best for critical thinking and concept exploration

## Common Learning Activities

| Activity | Description | Best Modes |
|---|---|---|
| Think-Pair-Share | Individual → partner → class | Discussion, collaboration |
| Gallery Walk | Rotating stations viewing posted work | Experiential, collaboration |
| Jigsaw Reading | Expert groups → teaching groups | Collaboration, guided |
| Exit Ticket | End-of-lesson written check | Feedback & reflection |
| Four Corners Debate | Physical movement to position | Discussion, experiential |
| Concept Mapping | Visual organisation of connections | Independent, explicit |
| Fishbowl Discussion | Inner/outer circle observation | Discussion, demonstration |
| Quick Write | Timed focused writing | Independent, feedback |
| Role Play | Character adoption in scenario | Experiential, demonstration |
| Experiment | Hands-on investigation and analysis | Experiential, guided |

## Learning Modes

Resources can incorporate these modes:

| Mode | Indicators |
|---|---|
| `collaboration` | Shared goals, co-operative problem-solving, peer tutoring, co-creation |
| `discussion` | Idea sharing, critical thinking, questioning, equitable participation |
| `feedback & reflection` | Goal tracking, self-assessment, peer feedback, iterative improvement |
| `guided` | Scaffolded learning, expert instruction, progressive independence |
| `explicit` | Direct instruction, teacher modelling, structured teaching, clear goals |
| `demonstration` | Student presentations, peer teaching, sharing created products |
| `experiential` | Hands-on making, exploring, investigating, design thinking |
| `independent` | Self-regulation, personal goal setting, deliberate practice, deep engagement |

Multiple modes can — and often should — be present in a single resource.

## Strategy Selection Principles

Choose strategies that:
1. Align directly with the learning objectives
2. Match the target year/grade level
3. Suit the resource type (unit, lesson, activity)
4. Fit the subject matter complexity
5. Work within the available time

**Focus on 2–3 strategies done well** rather than attempting to incorporate all of them. Quality of implementation matters more than quantity.
